# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its Code of
Conduct. The full text is published here:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating — opening issues, sending pull requests, or taking part in any
project space — you agree to uphold it.

## Enforcement

Report unacceptable behavior privately to the maintainers at
`security@greenrosesystems.com` — the same confidential channel used for
security disclosures (see [SECURITY.md](./SECURITY.md)). All reports are
reviewed and handled confidentially. Maintainers are obligated to respect the
privacy and safety of anyone who reports an incident.
