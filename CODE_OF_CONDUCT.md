# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). By taking part — issues, pull requests, any project space — you agree to uphold it.

Report unacceptable behavior privately through a [GitHub Security Advisory](https://github.com/laffeyp/cascade-img/security/advisories/new) (the same confidential channel as [security disclosures](./SECURITY.md)). Reports are handled confidentially.
